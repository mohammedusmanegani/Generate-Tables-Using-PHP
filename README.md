# Generate-Tables-Using-PHP

```ruby
<?php
$start = 1;
echo "Enter the table number : ";
$tableNumber = readline();
echo "Enter the limit of Table : ";
$tableLimit = readline();
for ($i = $start; $i <= $tableLimit; $i++) {
    echo $tableNumber . " X " . $i . " = " . $tableNumber * $i . "\n";
}

```

# Run the PHP file on cmd Line

```ruby
php tables.php
```
